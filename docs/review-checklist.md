# レビューチェックリスト

このチェックリストは `AutoMosaic-Review/Review Checklist.md` として reviewer に渡す前提です。

## 最初に確認すること
- `AutoMosaic-Review/` を開ける
- `Launch Auto Mosaic Review.cmd` から起動できる
- 起動チェックが意味の分かる内容で表示される
- 必要モデルが足りない場合に `必要モデルを取得` で解消できる

## 基本操作
- `動画を開く` から MP4 を選べる
- 中央プレビューに先頭フレームが表示される
- 再生 / 一時停止ができる
- 1秒戻る / 1秒進む ができる
- 1フレーム戻し / 1フレーム送り ができる

## AI 検出
- `AI検出開始` ボタンがすぐ見つかる
- 動画を開く前は押せない
- 動画を開いた後は押せる
- 実行中の状態が分かる
- 検出後にマスクが見える

## 編集
- トラックを選べる
- キーフレームを選べる
- inspector で軽い編集ができる
- canvas 上で楕円またはポリゴンを軽く動かせる
- 元に戻す / やり直し ができる

## 保存
- project を保存できる
- project を再読込できる
- 保存状態が `保存済み / 未保存の変更あり / 自動保存中 / 自動保存に失敗` で分かる

## 書き出し
- `書き出し` ボタンが見つかる
- 書き出し設定を変えられる
- 書き出しが完了する
- 実行中 / 待機中 / 最近の結果 が分かる

## 補足観点
- `検出: GPU / CPU` と `書き出し: CPU` の表示が意味どおりに見える
- reviewer が Git / npm / Python / 手動モデル配置を要求されない
