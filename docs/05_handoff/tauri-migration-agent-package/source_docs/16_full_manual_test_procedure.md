## 1. 前提・準備

### 動作確認用素材

| 素材 | 要件 | 用途 |
|------|------|------|
| **短尺動画 A** | 30秒〜1分 mp4、顔/人物含む、30fps 推奨 | 基本操作・検出テスト |
| **短尺動画 B** | 別の動画（できれば複数人物） | 複数トラックテスト |
| **長尺動画**（任意） | 3分以上 | 書き出し・パフォーマンステスト |

### 環境要件

- Windows 11、Python 3.11+
- 検出モデル導入済み（`models/320n.onnx` 最低限）
- FFmpeg インストール推奨（音声付き書き出しに必要）

### 起動手順

```bash
# 初回
setup.bat

# 通常起動
run_ui.bat
# または
python -m app.main
```

### 全体所要時間

- **全項目通し**: 約 2〜3 時間
- **重要項目のみ（★マーク）**: 約 1 時間

### チェック記法

- `[ ]` 未確認
- `[x]` 確認済み・OK
- `[!]` 問題あり（要修正、末尾の不具合報告テンプレートに記録）

## 2. 基本機能

**目的**: アプリの根幹となる動画読込・表示・再生・保存が正常動作することを確認する。
**所要**: 約 15 分

### 2-A) 起動・初期状態 ★

| # | 操作 | 期待結果 |
|---|------|---------|
| 1 | `run_ui.bat` ダブルクリック | アプリが起動する。エラーダイアログなし |
| 2 | 初期画面を目視確認 | 3ペイン構成: 左パネル + 中央プレビュー + 右パネル + 下タイムライン + ステータスバー |
| 3 | ステータスバーを確認 | フレーム番号・タイムコード・解像度・FPS が表示される（動画未読込なので `-` や空欄可） |
| 4 | ダークテーマが適用されている | 背景が暗色（`#141414` 系）、テキストが明色（`#E0E0E0` 系） |

- [ ] 起動直後にクラッシュしない
- [ ] 3ペイン＋タイムライン＋ステータスバーのレイアウトが表示される
- [ ] ダークテーマが適用されている
