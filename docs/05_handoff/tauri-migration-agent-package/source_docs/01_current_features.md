# 現在できること

この文書は、今の Auto Mosaic で実際に使える機能を、人間向けに整理したものです。

## 画面の中心

- PySide6 ベースのデスクトップアプリとして動作する
- 左にトラック一覧、中央にプレビュー、右にプロパティパネル、下にタイムラインがある
- 編集はプレビューとタイムラインを見ながら進める

## 動画の読み込みと検出

- 動画を開ける
- 自動検出を実行できる
- 選択区間だけを検出できる
- 現在フレームだけを検出できる
- 対象が見えなくなったあとも、トラックをすぐ消さない設計になっている

## 編集まわり

- プレビュー上でマスクを直接調整できる
- 楕円マスクと多角形マスクを扱える
- キーフレームを追加、削除、複製できる
- タイムラインでフレーム単位の調整ができる
- トラックが増えてもタイムラインを縦スクロールで全行に到達できる
- トラックの表示 / 非表示を切り替えられる
- トラック単位で書き出し対象から除外できる（表示と書き出しは独立）
- 書き出し対象外のトラックはプレビュー・タイムラインで視覚的に区別される
- 右クリックメニューからトラック操作を行える

## 取り消しと確認

- Undo / Redo が使える
- 現在の操作名や件数が分かるようにしてある
- 書き出し前に危険フレームを確認する導線がある

## 書き出し

- 書き出し設定を選べる
- 書き出しジョブをキューに入れられる
- 起動時に未完了ジョブを復元できる
- 書き出し中は進捗を確認できる
- 途中キャンセルに対応している

## 復元と保全

- クラッシュリカバリがある
- 保存されていない編集状態を復元しやすい
- 起動時に必要な確認を行う導線がある

## GPU と推論

- GPU / 推論設定を変更できる
- GPU が使えない場合は CPU にフォールバックする
- 検出バックエンドは切り替え可能
- 現在は NudeNet と EraX を検出バックエンドとして扱える
- SAM2 tiny は高品質な輪郭生成モードで利用できる
