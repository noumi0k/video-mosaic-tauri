# Auto Mosaic レビューガイド

> 5月レビュー向け操作ガイド（2026-04-13 作成）

---

## はじめに

Auto Mosaic は、動画の特定領域を自動検出し、モザイク処理を行うデスクトップアプリです。  
検出エンジンを差し替えることで、顔・文字・ナンバープレートなど様々な対象に対応できます。

このガイドでは、レビュー時に辿る操作手順を説明します。

---

## 1. 起動

### Windows

1. `setup.bat` をダブルクリック（初回のみ）
   - Python 3.11 以上が必要です
   - 仮想環境の作成と依存パッケージのインストールが行われます
2. `run_ui.bat` をダブルクリック
   - アプリが起動します
   - 初回は仮想環境が未作成の場合、自動的に `setup.bat` が呼ばれます

### 起動に失敗する場合

- Python が PATH に入っているか確認してください
- `scripts/check_env.py` で環境診断ができます

---

## 2. 検出エンジンの確認

起動直後、ツールバーに以下が表示されます：

```
[検出エンジン:] [▼ NudeNet 320n] [対象カテゴリ ▾] [詳細設定...]
```

コンボボックスには、導入済みのモデル名（例: `NudeNet 320n` / `NudeNet 640m` / `EraX v1.1`）が並びます。

- **検出エンジン**: 現在使用中のモデルを選択・切替できます
- **対象カテゴリ**: 検出する対象の種類を選べます（チェックボックス）。選択中モデルで対応していないカテゴリはメニューから自動的に除外されます
- **詳細設定...**: 推論解像度・検出閾値・デバイス設定など高度な設定（検出エンジン選択はツールバーのみが主導線です）

ステータスバー（画面下部）にも現在のモデル名が常時表示されます。

### 検出モデル管理ダイアログ

「設定 > 検出モデル管理...」メニューから、既知モデルの導入状況を一覧で確認できます。

- 対象モデル: `NudeNet 320n` / `NudeNet 640m` / `EraX v1.1`
- 各モデルについて「モデル名 / 利用可否 / モデルファイルパス / 対応対象カテゴリ」を表示
- 利用可否は `利用可能` / `変換が必要` / `モデルファイルなし` の3状態
- 利用可能な別モデルを選択して「選択したモデルに切替」ボタンで切替可能
- EraX のみ `.pt` しか無い場合に「変換が必要」となり、`ultralytics` があれば「ONNX に変換」ボタンから `.onnx` を生成できます（EraX 専用導線）
- ツールバーのエンジン切替と同期します

SAM2 は輪郭エンジンのため、このダイアログには表示されません。モデル追加ウィザード・公式配布ページ一覧・未知モデルのラベル対応編集・任意 `.pt` の汎用変換は5月レビュー版では実装していません。

---

## 3. 検出エンジンの導入

検出モデルは本体に同梱していません。ユーザー自身で入手・配置する設計です。  
5月レビュー版では利便のため `setup.bat` がモデル取得を試みますが、これはレビュー用の暫定動作であり、製品版の恒久導線としては扱いません。  
詳細は [モデル導入手順](02_model-setup-guide.md) を参照してください。

モデル未導入の場合でも、アプリの起動と手動操作（マスクの手動追加・編集・書き出し）は可能です。検出モデル管理ダイアログを開くと、どのモデルが未導入かを確認できます。

---

## 4. 動画を開く

1. メニュー「ファイル > 動画を開く...」または `Ctrl+O`
2. MP4 / MOV / MKV / AVI ファイルを選択
3. プレビューにフレームが表示され、タイムラインにフレーム数が表示されます

---

## 5. 自動検出

1. ツールバーの「自動検出 ▾」をクリック
   - **全区間検出**: 動画全体をサンプリングして検出
   - **選択区間のみ検出**: タイムラインで `I`（開始）/ `O`（終了）を設定した範囲のみ検出
2. ツールバーの「フレーム検出」で現在フレームのみ検出も可能
3. 検出が完了すると、左パネル「トラック一覧」にトラックが追加されます

---

## 6. マスク編集

- **移動**: マスクをドラッグ
- **リサイズ**: ハンドルをドラッグ（楕円は四隅/辺、多角形は頂点）
- **頂点追加**: 多角形の辺をダブルクリック
- **頂点削除**: 頂点を Alt + 右クリック
- **キーフレーム追加**: `K` キー
- **キーフレーム削除**: `Shift+K`
- **フレーム移動**: `←` / `→`（1フレーム）、`Shift+←` / `Shift+→`（10フレーム）

右パネル「エフェクトコントロール」でモザイク強度・拡張幅を調整できます。

---

## 7. トラックの書き出し対象制御

各トラックには「書き出し対象 ON/OFF」があり、モザイク除外を制御できます。

- 左パネルのトラック行にある「書出」/「除外」ボタンで切替
- 除外トラックはプレビューで半透明のドット線で表示されます
- 書き出し時に除外トラックのモザイクは適用されません

---

## 8. モザイクプレビュー

- `M` キーでモザイクプレビューの ON/OFF を切替
- 実際のモザイク処理結果をリアルタイムで確認できます

---

## 9. 書き出し

1. メニュー「書き出し > 動画を書き出す...」または `Ctrl+M`
2. 書き出し設定ダイアログで以下を設定：
   - 出力先ファイルパス
   - 解像度（元解像度 / 720p / 1080p / 4K）
   - コーデック・ビットレート
3. 「書き出し」をクリック
4. 進行状況はステータスバーに表示されます

---

## 10. プロジェクト保存

- `Ctrl+S` でプロジェクト（トラック・設定）を JSON ファイルに保存
- `Ctrl+Shift+O` で保存済みプロジェクトを読み込み

---

## 主要ショートカット

| キー | 機能 |
|------|------|
| `Ctrl+O` | 動画を開く |
| `Ctrl+S` | プロジェクトを保存 |
| `Space` | 再生 / 停止 |
| `←` / `→` | 1フレーム移動 |
| `K` | キーフレーム追加 |
| `M` | モザイクプレビュー ON/OFF |
| `Ctrl+Z` / `Ctrl+Y` | Undo / Redo |
| `Ctrl+M` | 書き出し |
| `I` / `O` | 区間開始 / 終了マーカー |
| `N` | 楕円トラック追加 |
| `Shift+N` | 多角形トラック追加 |

全ショートカットは `F1` キーで確認できます。
