# 既知の制限事項

> 5月レビュー時点（2026-04-13 作成）

以下は現在のバージョンにおける既知の制限です。  
「バグ」ではなく「未実装」または「設計上の制約」です。

---

## 検出エンジン関連

| 項目 | 状態 | 備考 |
|------|------|------|
| 検出モデルの同梱 | 非同梱（設計方針） | 本製品はモデル差し替え型の汎用モザイク編集ソフトとして設計されており、検出モデルは同梱しません。ユーザー自身が入手・配置します |
| `setup.bat` のモデル自動取得 | レビュー版暫定 | 5月レビュー時の利便のために残していますが、製品版UIの恒久導線ではありません。失敗時は手動配置が必要です |
| 検出モデル管理ダイアログ | 最低限版を実装 | 既知モデルの一覧・利用可否・ファイルパス・対応カテゴリ・現在選択の表示と切替に限定します |
| UI からのモデル追加（ウィザード） | 未実装 | `models/` フォルダへの手動配置で対応してください |
| モデルの再スキャン | 未実装 | モデルを追加した場合はアプリの再起動が必要です |
| 公式配布ページ一覧 UI | 未実装 | 将来バージョンで検討（ユーザーがブラウザで開いて自分で取得する導線に限定） |
| アプリ内からの直ダウンロード | 方針上実装しない | モデルのライセンス適法性・互換性は保証対象外のため、アプリ内での直ダウンロード・ワンクリック取得・ミラー配布は行いません |
| 未知モデルのラベル対応編集 UI | 未実装 | 既知モデル (NudeNet 320n / 640m, EraX v1.1) のみ対象カテゴリを保証します |
| 学習状態ランプ / 教師データ保存 UI | 未実装 | 現時点では検出データの教師化・学習機能は提供しません |
| EraX `.pt` → `.onnx` 変換 | EraX 専用のみ実装 | `EraXAdapter` は `.onnx` のみ読み込みます。`.pt` しか無い場合は管理ダイアログの変換ボタン（`ultralytics` 必須）で `.onnx` を生成します。任意 `.pt` の汎用変換は未実装・非対応です |

## トラッキング関連

| 項目 | 状態 | 備考 |
|------|------|------|
| 後ろ向き追跡（backward tracking） | Phase 1（UI 導線のみ） | 未検出帯から区間検出を起動する UI 導線はあります。自動的な後方伝播は未実装です |
| トラック分割の Undo | 制限あり | 分割は取り消せますが、分割後に両方のトラックを編集した場合は完全な復元が困難な場合があります |

## 編集関連

| 項目 | 状態 | 備考 |
|------|------|------|
| Undo 履歴ツリー表示 | 未実装 | Undo/Redo 自体は動作します（最大 100 操作） |
| トラック順序のドラッグ並べ替え | 未実装 | トラック一覧の表示順は生成順です |
| 差分オーバーレイ | 実験的機能 | 表示メニュー + Shift+M からアクセス可能。ツールバーからは撤去されています |

## 書き出し関連

| 項目 | 状態 | 備考 |
|------|------|------|
| WebM / AVI 出力 | 動作するが品質に制約 | MP4 (H.264) が推奨出力形式です |
| FFmpeg 未インストール時 | OpenCV ライタにフォールバック | 音声なし・コーデック制約あり。FFmpeg の導入を推奨します |

## その他

| 項目 | 状態 | 備考 |
|------|------|------|
| macOS / Linux 対応 | 基本動作 | Windows での動作を主にテストしています |
| 教師データ保存・再学習 | 未実装 | 将来バージョンで対応予定です |
| インストーラ / アップデータ | 未実装 | 現在は手動セットアップです |
| 画像（静止画）のモザイク処理 | 未対応 | 動画のみ対応しています |
