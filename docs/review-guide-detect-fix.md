# Auto Mosaic 再・非技術者レビュー手順（detect修正版）

## 目的
前回の detect failure 修正後に、
専門知識がない人でも
「起動 → 動画読込 → AI検出 → 最低限編集 → 保存 → 書き出し」
まで通せるかを確認する。

## レビュー対象者
- 開発者ではない人
- このソフトを初めて触る人が望ましい
- 1〜3人いれば十分

## 実施環境
- できれば開発環境ではない Windows PC
- review package 一式
- テスト用動画 1本

## 今回の確認項目

### 1. 起動
- Launch Auto Mosaic Review.cmd から起動できたか
- checklist が止まらず進むか
- Ready 相当の状態まで迷わず見られるか

### 2. 動画読込
- 動画を開けるか
- プレビューが表示されるか
- どこを押せばよいか直感的に分かるか

### 3. 最初の保存
- project 保存ができるか
- 保存できたことが分かるか

### 4. AI検出
- AI検出を開始できるか
- 進捗や待ち状態が分かるか
- エラーにならず結果が返るか
- 検出後、マスクやトラックが見えるか

### 5. 最低限編集
以下のどれか1つでよい
- track 選択
- keyframe 選択
- inspector 変更
- canvas 上で少し動かす
- timeline 上で少し触る

確認点:
- 何を選んでいるか分かるか
- 編集結果が反映されるか
- 操作に迷いすぎないか

### 6. 保存して再読込
- 編集後に保存できるか
- 再度開いて内容が残るか

### 7. 書き出し
- export を追加できるか
- 進捗表示が理解できるか
- 結果ファイルが作られるか
- どこに保存されたか分かるか

## 記録してほしいこと

### 必須
- どこで迷ったか
- どこが分かりにくかったか
- どこで止まりそうになったか
- 一番直してほしい点

### 任意
- 見た目の印象
- 動作速度への不満
- 用語が難しい箇所

## 判定基準

### 今回OK
- 起動できる
- 動画を開ける
- AI検出が通る
- 最低限編集ができる
- 書き出せる

### 今回NG
- detect が再び途中死する
- 保存が分からない
- 何を選択しているか分からない
- 書き出し結果の場所が分からない
- 非技術者が一人で通せない
